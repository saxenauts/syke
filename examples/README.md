# Examples

| File | What It Does |
|------|-------------|
| `quick_start.sh` | Full CLI pipeline: setup (detect, collect, build profile, output) |
| `mcp_config.json` | MCP server config for Claude Code settings |

## Quick Start

```bash
chmod +x quick_start.sh
./quick_start.sh your-username
```

## MCP Setup

Copy `mcp_config.json` into your `~/.claude/settings.json` under `mcpServers`, updating the paths.
