## Quick Setup

<Steps>

### Clone and install

```bash
git clone https://github.com/saxenauts/syke.git && cd syke
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
```

### Run tests

```bash
python -m pytest tests/ -v
```

293 tests across the suite. Core tests are mocked and run without external credentials.

### Make changes

Pick an area to contribute and open a PR.

</Steps>
