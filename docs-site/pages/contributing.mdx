## Quick Setup

<Steps>

### Clone and install

```bash
git clone https://github.com/saxenauts/syke.git && cd syke
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
```

### Run tests

```bash
python -m pytest tests/ -v
```

304 tests across 20 files, all mocked. No API key needed for core tests (some optional tests require `ANTHROPIC_API_KEY`, skipped in CI without it).

### Make changes

Pick an area to contribute and open a PR.

</Steps>
