# Syke — Agentic Memory

Your digital footprint — code, conversations, commits, emails — synthesized into a living model of who you are. Runs as a daemon. Feeds Claude Code, Claude Desktop, Codex, Cursor, Windsurf, Hermes, and any CLI workflow.

<Callout type="info">
Current release: v0.4.5. 293 tests. 5 providers (`claude-login`, `codex`, `openrouter`, `zai`, `kimi`).
</Callout>

## Get Started

One command. Setup opens an interactive provider picker, ingests available sources, starts the daemon, and distributes your memex to connected tools.

```bash copy
uvx syke setup
```
